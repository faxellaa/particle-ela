# Particle Hand Tracking 🧤✨

Proyek interaktif berbasis Web (HTML/CSS3/JS) yang memungkinkan partikel di layar mengikuti gerakan tangan secara live menggunakan webcam.

### 🚀 Cara Menjalankan
1. Pastikan sudah menginstall **VS Code** terbaru.
2. Buka VS Code, lalu pilih **Open Folder** dan arahkan ke folder `particle-ela`.
3. Pilih file `.html`.
4. Klik kanan pada file tersebut, lalu pilih **Open with Live Server** (atau cukup buka file .html di Google Chrome).

### 🛠️ Teknologi
* **HTML5 & CSS3**
* **JavaScript** (Vanilla / Canvas API)
* **MediaPipe Hands** (Untuk deteksi tangan lewat browser)

### 💡 Tips
Pastikan kamu memberikan izin (Allow) akses kamera saat browser memintanya agar fitur hand tracking bisa berjalan.
